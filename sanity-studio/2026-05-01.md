### 6. Receipt Schema — Withholding Tax (WHT)
Added to Group 3 (Amounts & Payment):

| Field | Title | Notes |
|-------|-------|-------|
| `withholdingTaxRate` | 3.9 · WHT Rate | 1% / 1.5% / 3% / 5% / 10% / Custom |
| `whtAmount` | 3.10 · WHT Amount Deducted by Payer | Auto-calculated via `ReceiptWhtAmountInput` |
| `netReceivedSummary` | 3.11 · Net Received | Total − WHT via `NetReceivedSummary` |
| `whtCertNumber` | 3.12 · WHT Certificate No. | ใบหักภาษี ณ ที่จ่าย cert number |
| `whtDocs` | 3.13 · WHT Certificate Documents | File upload array |

New components created:
- `components/ReceiptWhtAmountInput.tsx` — mirrors `AutoWhtAmountInput` but reads `totalAmount`
- `components/NetReceivedSummary.tsx` — displays Total − WHT = Net Received

---

### 7. Receipt Schema — VAT Type
Added `vatType` (3.2) between subtotal and vatAmount:

| Value | Meaning |
|-------|---------|
| `exclusive` | VAT added on top of subtotal |
| `inclusive` | VAT included in price |
| `zero` | 0% VAT |
| `none` | No VAT |

- `vatAmount` (3.3) is now read-only when `vatType` is `none` or `zero`
- All subsequent fields renumbered: 3.4 Total Amount, 3.5 Currency, 3.6 Method, 3.7 Date, 3.8 Bank Ref
